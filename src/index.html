O nás

EASYCON Solution je technologická firma, která vyvíjí software a integruje hardware pro oblast průmyslového využití. Zabýváme se především IoT (internet věcí) a vývojem webových aplikací.

Principem naší firmy je poskytování co nejvyšší kvality. Flexibilita a stálá snaha o inovovativní řešení zaručuje maximální vyhovění požadavkům klienta.
